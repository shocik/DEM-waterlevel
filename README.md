# DEM-waterlevel

Repozytorium zawiera aktualnie rozwijany projekt, w ramach którego tworzony jest model uczenia maszynowego, pozwalający na odszumianie obszarów rzek na fotogrametrycznych cyfrowych modelach terenu oraz bezpośrednie uzyskanie wartości wysokości wysokości powierzchni wody wyrażonej w metrach nad poziomem morza.

Poniżej znajduje się wizualizacja pojedynczej próbki datasetu. Pierwszy obraz to zdjęcie true color (3 kanały RGB), drugi to uzyskany fotogrametrycznie cyfrowy model terenu (1 kanał zawierający wartości mnpm), natomiast trzeci to odszumiony cyfrowy model terenu (również 1 kanał zawierający wartości mnpm).
